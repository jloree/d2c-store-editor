# D2C Store Editor

A web-based tool for designing and customizing Direct-to-Consumer store layouts with real-time preview.

## Features

- **Live Preview**: See changes instantly as you customize
- **Multiple Page Types**: Home page, Product details, Cart views
- **Font Customization**: 80+ fonts including Adobe Fonts, Google Fonts
- **Color Theming**: Complete color palette control
- **Layout Options**: 3 or 4 column grid layouts
- **Advanced Overrides**: Granular control over individual elements
- **Code Generation**: Export HTML & CSS for implementation
- **Save/Load**: Store and share configurations

## Usage

1. Open `index.html` in your browser
2. Use the left panel to customize your store design
3. See live preview on the right
4. Generate HTML/CSS code when ready
5. Save/load configurations for reuse
